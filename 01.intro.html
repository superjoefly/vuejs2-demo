Vuejs Intro

Declarative Rendering via Text Interpolation

Render a message using string interpolation:


      <div id="app">

        {{ message }}

      </div>

      -----------------

      var app = new Vue({
        el: '#app',
        data: {
          message: 'Well, Hello There!'
        }
      })

Message: {{ message }}

Entering app.message = 'something else' in the console will change the value of the message property!

Binding Element Attributes

Dynamically bind the title attribute to a data property using v-bind:


      <div id="app2">

        <span v-bind:title="message2">Hover over me to see the title!</span>

      </div>

      -------------------

      var app2 = new Vue({
        el: '#app2',
        data: {
          message2: 'Here is the title!'
        }
      })
Hover over me to see the title!

Entering app2.message2 = 'Something else' in the console will change the value of the dynamically bound title!

Note: the shorthand for v-bind: is to just use the colon : So in the example above we would just use :title instead of v-bind:title

Conditionals and Loops

Toggle the presence of an element:


      <div id="app3">

        <p v-if="seen">This is a paragraph!</p>

      </div>

      ---------------------

      var app3 = new Vue({
        el: '#app3',
        data: {
          seen: true
        }
      })

This is a paragraph!

Entering app3.seen = false will toggle the presence of the element!

Display a List of Items

Display a list of todo items using v-for:


      <div id="app4">

        <ul>

          <li v-for="todo in todos">

            {{ todo.text }}

          </li>

        </ul>

      </div>

      -------------------------

      var app4 = new Vue({
        el: '#app4',
        data: {
          todos: [
            { text: 'Study JavaScript' },
            { text: 'Practice Vuejs' },
            { text: 'Walk Bella!' }
          ]
        }
      })
  • {{ todo.text }}

Entering app4.todos.push({text: 'something else'}) will append an item to the todo list!

Respond to User Input

Use event listeners to invoke methods on the Vue instance:


      <div id="app5">

        <button v-on:click="reverseMessage">Reverse Message</button>

        <p>{{ message }}</p>

      </div>

      --------------------------

      var app5 = new Vue({
        el: '#app5',
        data: {
          message: 'Reverse this message!'
        },
        methods: {
          reverseMessage() {
            this.message = this.message.split('').reverse().join('');
          }
        }
      })

{{ message }}

In the method above, we update the state of our app without touching the DOM - all DOM manipulations are handled by Vue.

Note: the shorthand for v-on: is to just use the @ symobol. So in the example above we would just use @click instead of v-on:click

Two Way Binding

Use v-model to apply dynamic, two-way binding for inputs:


      <div id="app6">

        <input type="text" v-model="message" />

        <br />

        <p>{{ message }}</p>

      </div>

      ------------------------

      var app6 = new Vue({
        el: '#app6',
        data: {
          message: 'Change Me!'
        }
      })

{{ message }}

Note: v-model is actually short-hand for :value="dataProperty" and @input="dataProperty = $event.target.value"

Register and Render Components

In Vue, we can compose and register components that are small, self-contained and reusable:


      <div id="app7">

        <my-component></my-component>

      </div>
      --------------------------
      Vue.component('myComponent', {
        template: `
          <div>Here is my custom component!</div>
        `
      })

      var app7 = new Vue({
        el: '#app7'
      })

This is a very simple example of component usage.

Parent Child Component Communication

Demonstrate passing data from the parent scope into child components using props:


      <div id="app8">

        <ol>

          <!-- Render todo items -->

          <todo-item

            v-for="item in todos"

            :todo="item"

            :key="item.id"

          ></todo-item>

        </ol>

      </div>

      --------------------------
      // Vue Component
      Vue.component('todo-item', {
        template: `
          <li>{{ todo.text }}</li>
        `,
        // Accept prop from parent
        props: ['todo']
      })

      // Vue Instance
      var app8 = new Vue({
        el: '#app8',
        data: {
          todos: [
            {id: 0, text: 'Go to the store'},
            {id: 1, text: 'Go to work'},
            {id: 2, text: 'Walk the dog'},
          ]
        }
      })

Using components separates the app into smaller units and helps keep development manageable.
